在自己的github主页放点自定义内容

|字数总计:111|阅读时长:1分钟|阅读量:

在自己的github主页放点自定义内容

创建仓库

首先，点击“New repository”创建一个仓库

Repository填自己用户名，就是隔壁那个owner的名字

勾选“Add a README file”

点击“Create repository”

创建之后，点击右边的“Edit README”，在里面编辑好内容保存即可。

大功告成

如果想进一步美化个人主页，还可以尝试使用HTML、CSS和JavaScript。

Enjoy it, 下次见~
